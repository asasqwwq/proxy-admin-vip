# proxy-admin-vip
proxy-admin vip 版本（功能和免费版一致，服务数量无限制），首次使用，会在下面的目录生成id.txt文件，把里面的内容绑定到授权平台即可。  
Windows：C:\gpa\id.txt  
Linux & MacOS : /etc/gpa/id.txt
